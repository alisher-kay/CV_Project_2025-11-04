## ğŸ¨ Customization

To customize this CV for your own use:

1. **Update Personal Information**
   - Edit the name, job title, and contact details in `index.html`
   - Replace the profile image in the `img/` folder

2. **Modify Skills**
   - Update the skills lists in the left sidebar
   - Add or remove technical skills, frameworks, and tools

3. **Update Experience**
   - Edit work experience entries in the right column
   - Modify company names, dates, and job descriptions

4. **Styling Changes**
   - Customize colors, fonts, and layout in `styles.css`
   - Adjust responsive breakpoints if needed
